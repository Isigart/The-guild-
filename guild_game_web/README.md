# Compagnie de Mercenaires — Web

Jeu RPG de gestion de guilde. Version Web.

## 🚀 Déploiement sur GitHub Pages (depuis mobile)

### 1 — Créer le repository GitHub

1. Aller sur **github.com**
2. **New repository** → nommer `guild-game` → **Public** → **Create**

### 2 — Uploader le projet

1. Dans le repo → **Add file → Upload files**
2. Uploader le contenu du zip `guild_game_web_v1.zip`
3. **Commit changes**

### 3 — Activer GitHub Pages

1. **Settings → Pages**
2. Source : **GitHub Actions** → **Save**

### 4 — Attendre le build

- Onglet **Actions** → build ~3-5 minutes ✅

### 5 — Jouer

```
https://TON_USERNAME.github.io/guild-game/
```
